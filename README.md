# 3D Print Service System

Web-based system developed for creating professional service orders for 3D printing with automatic PDF export.

## Features

- Client registration
- RG/CPF registration
- Address field
- Service date
- Start and end time
- Service value
- Detailed service description
- Piece photo upload
- Responsible technician registration
- CREA field
- Automatic PDF generation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jsPDF

## Objective

The goal of this project is to simplify the creation of professional service orders for 3D printing jobs directly in the browser without requiring installation.

## How to Use

1. Open the `index.html` file
2. Fill in the service order information
3. Upload photos of the printed piece
4. Click `DOWNLOAD PDF`
5. The PDF will be generated automatically
